# vscode-markdown-tefcha

Add support for flowchar preview in Markdown file.

![demo](demo.gif)


## Usage

Use `tefcha` fenced code blocks

~~~
```tefcha

# Write pseudo code here

if morning
  hello
end

```
~~~
